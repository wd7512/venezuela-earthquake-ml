Contributing

This hub is a public resource. If you're a researcher, data scientist, or engineer who can help fill the gaps below, this page explains where to plug in.

Public-first: the pages visible in the main navigation (Overview, Intel, Gaps, Zones) are for responders and the public. This Contributing section is for people who want to contribute ML tooling.

How to Help

1. Pick a Gap

Visit the Gaps page for a prioritized list of where ML can add value. High-severity gaps are the most impactful starting points.

2. Claim or Open a Track

Below are six ML tracks. Each has data sources, suggested models, and milestones. If you want to work on one, open an issue or submit a PR with your proposed approach.

3. Run the Build

uv sync
uv run python build_site.py
# Output goes to dist/

Data Pipeline

The site data lives in data.json. A cron job regenerates it periodically from INTEL_SCOUTING.md and live sources. To update manually:

uv run python build_site.py    # regenerates charts + rebuilds all pages from data.json
