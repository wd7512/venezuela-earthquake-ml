Missing Persons Platforms

If someone you know is missing, search and post on these volunteer-run platforms:

  • Desaparecidos Terremoto Venezuela — Search and post missing persons with photos
  • Venezuela Te Busca — Report name, age, gender, location; searchable registry
  • Google Person Finder — Search or register missing persons

Verified active, June 27 2026.

Technical Volunteers

If you have skills in mapping, data science, or engineering, there are specific ways to help:

  • HOT-OSM mapping — Help map affected areas. Check the Tasking Manager
  • Satellite damage assessment — Compare before/after imagery. Copernicus EMS
  • ML / data science — See the Contributing page for active projects
